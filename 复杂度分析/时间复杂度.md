## Java
两字符串相加时间复杂度是O(m+n),m n 分别为两个字符串的长度，他是两个字符串逐个字符加起来(即enumeration枚举)，然后形成一个新的字符串  
最好使用StringBuider来拼接，在字符串很大的时候时间复杂度不会很高
